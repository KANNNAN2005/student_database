Student Management System

Description:
This is a simple web-based Student Management System using HTML, CSS, and JavaScript.

Features:
- Add student details: Name, Roll Number, Course
- Display student list in a table
- Delete student record

How to Use:
1. Open index.html in a browser.
2. Fill out the form and click 'Add Student' to add a new student.
3. Click 'Delete' to remove a student from the list.

Files:
- index.html: Main HTML file
- style.css: Styling for the app
- script.js: JavaScript logic
- README.txt: Project documentation
